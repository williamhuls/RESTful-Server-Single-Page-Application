Tools Used

Leaflet

API

The Leaflet API allowed us to have an interactive map of the St. Paul area on our page. Leaflet allowed us to overlay markers on the map, and it made for easy modifications.

Nominatum

API

The Nominatum API let us search for locations by name and retrieve those locations based on the longitudinal and latitudinal coordinates.

STP Crime

API

Our API, St. Paul Crime, allowed us to retrieve the crimes committed around St. Paul in various ways such as by crime type, location, etc. These allowed us to make the Leaflet map interactive.
